Contenu de la clé USB : 
* _README_.txt - Le présent fichier 
* python3_idle - Environnement d'exécution de Python (3.4) 
* notepad++ - Editeur de texte
* Notepad++ - Répertoire des exécutables Notepad++ 
* PythonScripts - Répertoire des scripts Python 
* VirtualBox - Exécutables de lancement de Linux Mint (LMDE) en machine virtuelle 
  (Linux Mint est installé sur cette clé qui est bootable.)
* WinPython-32bit-3.4.3.6 - Exécutables WinPython 

Répertoire PythonScripts : 
- cesar_thomas.py: encodage d'une chaîne de caractères selon un code César 
- rasec_thomas.py: tentative de décodage d'une chaîne de caractères codées par un code César 
- dawg.py: script de hashage d'un dictionnaire - utilisé par rasec_thomas.py 
- frenchwords.txt: dictionnaire de mots français 

Procédure de base pour l'édition/exécution des scripts Python : 
- lancer l'environnement Python en double-cliquant sur 'python3_idle' 
- patienter un peu jusqu'à l'affichage de la console Python (le Shell) 
- ouvrir un script du répertoire PythonScripts (File/Open ou Ctrl+O) : 
  . 'cesar_thomas.py' pour encoder une chaîne 
  . 'rasec_thomas' pour tenter de décoder une chaîne codée 
- EDITION - le script peut être édité depuis la nouvelle fenêtre qui s'ouvre 
- EXECUTION - une fois le script ouvert, lancer la commande Run (menu Run/Run Module ou touche F5) 
- le script s'exécute alors dans la fenêtre console  

N.B. On met fin à une session python par une commande "quit()" dans la console
